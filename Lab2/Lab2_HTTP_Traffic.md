# Lab 2 - HTTP Traffic Capture

**Enviroments**
- Windows 10 VM (Client)
- Ubuntu 24.04 VM (Apache Server)
- VMware Host-Only Network

**Tasks Completed**
- Installed Apache2 on Ubuntu VM
- Accessed Apache page from windows VM
- Captured HTTP traffic in Wireshark

**Observations**
- TCP handshake visible between Windows and Ubuntu
- HTTP GET request from Windows browser
- HTTP 200 OK response from Apache
- Clean Host-Only traffic, no internet noise

**Tools Used**
- Wireshark
- Windows 10 VM
- Ubuntu 24.04 VM
